# Nmap Network Scan Lab

## Objective
The purpose of this lab is to perform basic network reconnaissance using Nmap and identify live hosts, open ports, and services on a target system.

## Tools Used
- Nmap
- Kali Linux

## Scope
This lab was performed in a controlled environment for educational purposes.

## Steps Performed
1. Identified live hosts on the local network
2. Selected a target IP
3. Ran a basic port scan
4. Performed service/version detection
5. Documented findings with screenshots
